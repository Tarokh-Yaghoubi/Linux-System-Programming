
#   CMAKE
### CMake is a cross-platform Makefile generator
### CMake -> Build System Generator
### CPack -> Package Generator used to create platform specific installers
### CTest -> A test driver tool used to run regression test
### CDash -> A web application for displaying test results and performing continuous integration testing.

### We write CmakeLists.txt, cmake utility reads that file and creates a complete build-system
