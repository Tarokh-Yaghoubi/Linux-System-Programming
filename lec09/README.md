
## What we learn here

### Opening and Closing Files
`fopen`, `fclose`

### Reading/Writing one character at a time
`fgetc`, `fputc`

### Reading/Writing line by line
`fgets`, `fputs`

### Reading/Writing using formatted I/O
`fscanf`, `fprintf`

### File position functions
`fseek`, `ftell`, `fgetpos`, `fsetpos`, `rewind`
